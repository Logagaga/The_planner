# The Planner (Ray Node-Planning)

節點式規劃APP。使用者用拖曳節點的方式規劃日常任務與長期人生目標，數位任務節點可透過程式碼或 AI Agent 真正執行成工作流。

## 專案結構

- `docs/` — 技術與人類行為科學研究、系統架構設計文件
- `app/` — Phase 1 APP 原始碼（React + Vite + @xyflow/react），詳見 `app/README.md` 的執行方式

## 開發階段

- [x] Phase 1：純視覺化規劃（拖曳節點、連線、卡片、引導式問答）
- [ ] Phase 2：彈性 QA 回饋收集、使用者互動資料
- [ ] Phase 3：數位任務執行卡片完整清單
